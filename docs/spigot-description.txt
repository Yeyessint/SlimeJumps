[SIZE=5][B][COLOR=#48b463]✦ Features[/COLOR][/B][/SIZE]
[LIST]
[*][B]Jump Pads[/B] — named pads on any block, each with its own horizontal & vertical launch power.
[*][B]Flight Routes[/B] — step on a pad and fly along waypoints to any destination, with a particle trail, whoosh sound, full damage immunity in flight and arrival effects. Perfect for sending players from the lobby to game areas!
[*][B]Double Jump[/B] — lobby-style mid-air boost on the jump key, compatible with /fly from other plugins.
[*][B]Holograms[/B] — floating text above pads using native 1.21 text displays. No hologram plugin needed.
[*][B]Editor Wand[/B] — left click a block to create a pad, right click to remove it. No commands needed.
[*][B]Pads GUI[/B] — paginated menu of every pad; click one to teleport straight to it.
[*][B]Presets[/B] — create pads with one command: parkour, cannon, bounce — or define your own in the config.
[*][B]Per-pad everything[/B] — power, direction (free or fixed), sound, particle, cooldown, potion effect, action bar message and console command (rewards, server transfers…).
[*][B]Statistics[/B] — server-wide launch totals and your most used pads.
[*][B]Safe by default[/B] — anti-spam cooldowns, fall damage protection, per-world disabling, sneak-to-bypass for staff.
[*][B]Bilingual[/B] — English & Spanish included; add any language by dropping in one file.
[*][B]Legacy friendly[/B] — pads from SlimeJumps 1.x are migrated automatically; optional classic "every slime block launches" mode.
[/LIST]

[SIZE=5][B][COLOR=#48b463]✦ Quick start[/COLOR][/B][/SIZE]
[LIST=1]
[*]Drop [B]SlimeJumps-x.x.x.jar[/B] into your [ICODE]plugins/[/ICODE] folder and restart.
[*]Stand on a block and run [ICODE]/sj create spawn[/ICODE] — that block is now a jump pad!
[*]Want a flight route? [ICODE]/sj route create togames[/ICODE], fly the path adding points with [ICODE]/sj route addpoint togames[/ICODE], then link it: [ICODE]/sj setroute spawn togames[/ICODE].
[/LIST]

[SIZE=5][B][COLOR=#48b463]✦ Commands[/COLOR][/B][/SIZE]
Main command: [ICODE]/slimejumps[/ICODE] (aliases [ICODE]/sj[/ICODE], [ICODE]/jumppads[/ICODE])
[CODE]
/sj create <name> [power] [vertical]   Create a pad where you stand
/sj create <name> --preset <preset>    Create a pad from a preset
/sj wand                               Editor wand (left click = create, right click = remove)
/sj remove | rename | toggle <pad>     Manage pads
/sj list | info | tp | near | gui      Find & inspect pads
/sj stats                              Launch statistics
/sj setpower | setvertical <pad> <v>   Launch strength
/sj setdirection <pad> <look|here>     Free or fixed launch direction
/sj setsound | setparticle <pad> ...   Per-pad effects
/sj setcooldown <pad> <ms|default>     Per-pad cooldown
/sj seteffect <pad> <effect> [s] [lvl] Potion effect on use
/sj setmessage <pad> <text|none>       Action bar message on use
/sj sethologram <pad> <text|none>      Floating hologram above the pad
/sj setcommand <pad> <command|none>    Console command on use (%player%)
/sj setroute <pad> <route|none>        Link a pad to a flight route
/sj route create|addpoint|delpoint|remove|list|info
/sj reload                             Reload everything
[/CODE]

[SIZE=5][B][COLOR=#48b463]✦ Permissions[/COLOR][/B][/SIZE]
[CODE]
slimejumps.admin        All commands (default: OP)
slimejumps.use          Being launched by pads (default: everyone)
slimejumps.doublejump   Double jump, when enabled (default: everyone)
[/CODE]

Consejos para la publicación:
1. Las dos imágenes ([IMG]) apuntan a la carpeta assets/ del repo en GitHub;
   funcionan en cuanto merges el PR con los assets.
2. Sube el jar de la release v2.4.0 como archivo del recurso, o marca
   "external download" con el enlace a GitHub Releases.
3. Cuando publiques una versión nueva, usa la pestaña "Updates" de Spigot y
   pega la sección correspondiente del CHANGELOG.md.
4. Añade 2-3 capturas/GIFs in-game cuando puedas (un jugador saliendo
   disparado, una ruta de vuelo con la estela, la GUI abierta): las capturas
   reales son lo que más descargas genera.
